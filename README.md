# Ejecutar en desarrollo

1. Tener Nest CLI instaldo

```bash
npm i -g @nestjs/cli
```

2. Clonar el repositorio

3. Ejecutar

```bash
npm install
```

4. Levantar la base de datos

```bash
docker-compose up -d
```

## Stack Usado

- Mongo
- Nest JS
